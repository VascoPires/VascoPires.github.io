Launches an interactive WebGL visualization of 3D failure surfaces (FI = 1), including Tsai-Wu, Tsai-Hill, Hashin, Puck, von Mises, Tresca, and maximum stress/strain criteria. Use the didactic parameter controls to compare how each criterion changes with material properties and interaction terms.
